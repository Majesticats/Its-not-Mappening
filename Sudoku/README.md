# Sudoku Game
A web-based Sudoku game built for CSCI 265.  
It includes a playable 9×9 grid, puzzle generator, solver, conflict detection, highlights, and autosave.

## How to Run
Just open **index.html** in any browser.

## Features
- Interactive 9×9 grid  
- Conflict detection (row, column, box)  
- Highlights: selected cell, peers, same numbers  
- Puzzle generator (Easy / Medium / Hard)  
- Backtracking solver  
- Check if solved  
- Reset puzzle  
- On-screen keypad for mobile  
- Auto-save (localStorage)

## Tech Used
- HTML  
- CSS  
- JavaScript modules  
- LocalStorage

## Algorithms
- **Solver:** backtracking  
- **Generator:** full-solution → remove numbers while maintaining unique solution  
- **Validation:** row/col/box checks
